# Example Files

A collection of example files demonstrating various programming languages and file formats.

## Contents

This repository includes example files for:

- **Languages**: JavaScript, TypeScript, Python, Go, Lua, C#, Shell
- **Web**: HTML, CSS, SCSS, Vue
- **Data**: JSON, CSV, YAML, TOML
- **Text**: Markdown, plain text

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development environment:

```bash
npm run dev
```

This will start both the client and server concurrently.

### Available Scripts

- `npm run dev` - Start both client and server
- `npm run c` - Start client
- `npm run s` - Start server

## Project Structure

This workspace contains example files in various formats for reference and testing purposes.
